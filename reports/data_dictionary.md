# Data Dictionary

## Bluestock Mutual Fund Analytics Capstone

### Introduction

This document provides a brief description of all datasets used in the Bluestock Mutual Fund Analytics Capstone project. It explains the purpose of each dataset, the type of information stored in each column, and the source file from which the data was collected. The data dictionary serves as a reference guide for understanding and analyzing the project data.

---

## Dataset 1: Fund Master

**Source File:** 01_fund_master.csv

| Column Name | Data Type | Description                                              |
| ----------- | --------- | -------------------------------------------------------- |
| amfi_code   | Integer   | Unique code assigned to every mutual fund scheme         |
| scheme_name | Text      | Name of the mutual fund scheme                           |
| fund_house  | Text      | Name of the asset management company managing the fund   |
| category    | Text      | Category under which the mutual fund is classified       |
| plan        | Text      | Indicates whether the scheme is a Direct or Regular plan |

---

## Dataset 2: NAV History

**Source File:** 02_nav_history.csv

| Column Name | Data Type | Description                               |
| ----------- | --------- | ----------------------------------------- |
| amfi_code   | Integer   | Unique identifier of the mutual fund      |
| date        | Date      | Date on which NAV was recorded            |
| nav         | Decimal   | Net Asset Value of the fund for that date |

---

## Dataset 3: AUM by Fund House

**Source File:** 03_aum_by_fund_house.csv

| Column Name | Data Type | Description                             |
| ----------- | --------- | --------------------------------------- |
| fund_house  | Text      | Name of the fund management company     |
| aum_crore   | Decimal   | Total Assets Under Management in crores |

## Dataset 4: Monthly SIP Inflows

## Dataset 7: Scheme Performance

**Source File:** 07_scheme_performance.csv

| Column Name       | Data Type | Description                                       |
| ----------------- | --------- | ------------------------------------------------- |
| amfi_code         | Integer   | Unique fund identifier                            |
| scheme_name       | Text      | Name of the mutual fund scheme                    |
| fund_house        | Text      | Fund management company                           |
| category          | Text      | Category of the mutual fund                       |
| return_1yr_pct    | Decimal   | Return generated by the fund over one year        |
| return_3yr_pct    | Decimal   | Return generated by the fund over three years     |
| return_5yr_pct    | Decimal   | Return generated by the fund over five years      |
| expense_ratio_pct | Decimal   | Annual fee charged for managing the fund          |
| sharpe_ratio      | Decimal   | Measure of risk-adjusted return                   |
| alpha             | Decimal   | Additional return generated above the benchmark   |
| beta              | Decimal   | Measure of fund volatility compared to the market |

---

## Dataset 8: Investor Transactions

**Source File:** 08_investor_transactions.csv

| Column Name        | Data Type | Description                                                |
| ------------------ | --------- | ---------------------------------------------------------- |
| investor_id        | Integer   | Unique identification number for an investor               |
| transaction_date   | Date      | Date on which the transaction took place                   |
| amfi_code          | Integer   | Mutual fund scheme identifier                              |
| transaction_type   | Text      | Type of transaction such as SIP, Lumpsum, or Redemption    |
| amount_inr         | Decimal   | Transaction value in Indian Rupees                         |
| state              | Text      | State of the investor                                      |
| city               | Text      | City of the investor                                       |
| city_tier          | Text      | Classification of city based on population and development |
| age_group          | Text      | Investor age category                                      |
| gender             | Text      | Gender of the investor                                     |
| annual_income_lakh | Decimal   | Annual income of the investor in lakhs                     |
| payment_mode       | Text      | Method used for making the payment                         |
| kyc_status         | Text      | Indicates whether KYC verification is completed            |

---
